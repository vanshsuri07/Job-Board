# 💼 Job Board

A platform to post and search for developer jobs.

## 🚀 Tech Stack

- **Frontend**: React
- **Backend**: Node.js + Express

## ✨ Features

- 🔐 User authentication (login/signup)
- 📝 Post a new job
- 🔍 Search and filter jobs
- 📄 Upload resume
- ⚡ Fast and responsive interface
